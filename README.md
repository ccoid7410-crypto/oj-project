# Durunuri OJ

충북과학고 동아리에서 사용하기 위해 개발한 자체 온라인 저지 시스템입니다.

기존 HustOJ를 설치하는 대신, 온라인 저지의 내부 구조를 직접 구현하고 자유롭게 확장해 보고자 시작했습니다. 단순한 문제 제출과 채점을 넘어 문제 출제·검토, 대회 운영, 권한 관리, 커뮤니티, 동아리 홈페이지 연동 등 다양한 개발과 연구를 수행할 수 있는 기반을 만드는 것을 목표로 합니다.

현재 프로젝트는 개발 중이며, 기능과 구조가 계속 변경될 수 있습니다.
